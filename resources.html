Affordable Housing Programs

Comprehensive breakdown of available housing schemes in Malaysia

PR1MA

Active

Target Group: Middle-income households

Income Range: RM2,500 - RM15,000

Property Type: Apartments, Condominiums, Landed

Key Features:

  • Below market price
  • Strategic locations
  • Quality developments
  • Malaysian citizens only

Available Locations:

Kuala Lumpur Selangor Penang Johor

MyFirst Home

Active

Target Group: First-time homebuyers

Income Range: RM3,000 - RM10,000

Property Type: New residential properties

Key Features:

  • 100% financing available
  • Competitive interest rates
  • Flexible repayment terms
  • Government guarantee scheme

Available Locations:

Nationwide

PPR (Program Perumahan Rakyat)

Active

Target Group: Low-income households

Income Range: Below RM3,000

Property Type: Low-cost flats

Key Features:

  • Highly subsidized housing
  • Rental and ownership options
  • Basic amenities included
  • Community facilities

Available Locations:

Urban areas State-specific

Rumah Selangorku

Active

Target Group: Working in Klang Valley

Income Range: RM3,000 - RM10,000

Property Type: Serviced apartments

Key Features:

  • Subsidized housing prices
  • Various housing types
  • Integrated township development
  • Easy financing options

Available Locations:

Selangor
